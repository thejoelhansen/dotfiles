# Installation

git clone git@github.com:thejoelhansen/dotfiles.git ~/.dotfiles && ~/.dotfiles/install.sh

It will prompt you before replacing if the files already exist.

# Features

- `bin/command`: gist 

# Uninstall

```
unlink ~/.bashrc
unlink ~/.bin
unlink ~/.vim
unlink ~/.vimrc
unlink ~/.zsh
rm -rf ~/.dotfiles
```
