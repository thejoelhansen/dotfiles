# Installation

Run `bin/install` command to copy files over. It will prompt you before replacing if the files already exist.

# Features

- `bin/command`: gist 

# Uninstall

To remove the dotfile configs, run the following commands. 

```
unlink ~/.bin
unlink ~/.gitignore
unlink ~/.gitconfig
unlink ~/.gemrc
unlink ~/.gvimrc
unlink ~/.irbrc
unlink ~/.vim
unlink ~/.vimrc
rm ~/.zshrc # careful here
rm -rf ~/.dotfiles
```
